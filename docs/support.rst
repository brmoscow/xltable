Troubleshooting
---------------

Enable logging
^^^^^^^^^^^^^^

To diagnose issues, enable logging in ``settings.json``:

.. code-block:: json

   "WRITE_LOG": true

Log files are written to the ``xltable/log`` folder.
After enabling, restart the service and reproduce the issue, then inspect the log files.
