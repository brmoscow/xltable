XLTable Documentation
=====================

**Excel meets Big Data.**

XLTable is an OLAP server that connects **Excel Pivot Tables**
directly to modern analytical databases without CSV exports or BI tools.

It acts as a **semantic and security layer** between Excel and databases such as
ClickHouse, BigQuery and Snowflake.

Why XLTable?
============

- Use native Excel Pivot Tables on big data
- Centralized semantic layer (dimensions, measures, hierarchies)
- Fine-grained access control and Active Directory integration
- Query result caching for fast Excel experience
- Designed for self-hosted and cloud deployments

Supported data sources
======================

- ClickHouse
- BigQuery
- Snowflake
- Trino
